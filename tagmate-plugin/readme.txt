== Description ==

tagmate.io plugin helps you easily install third-party services code snippets on your website.


== Frequently Asked Questions ==

= How do I get started? =

This plugin only works with the information provided via the [tagmate.io](https://tagmate.io) widget that we provide to third-party service owners.

= Where can I find my Platform ID? =

You can copy your Platform ID from the tagmate widget instructions that we provide to third-party service owners.

= Do I also need a User ID? =

You only need a User ID for some specific platforms. You don't need it if it was not provided via the tagmate.io widget.

= Will this plugin also load the code snippet on my admin pages? =

Code snippets are only loaded for your theme/front-facing pages.

= Does this plugin work on older versions of WordPress or PHP? =

Possibly, but I've not tried. I can only provide support if you're using v3.9 WordPress and PHP 5.2.4 or newer.

== Installation ==

= WordPress Admin Method =
1. Go to you administration area in WordPress `Plugins > Add`
2. Search for `tagmate.io`
3. Click on Install and Activate the plugin
4. Open the plugin settings page `Settings > Tagmate`
5. Enter the information that you got from tagmate.io SaaS widget

= File Transfer Method =
1. Upload the complete `tagmate` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Open the plugin settings page `Settings > Tagmate`
5. Enter the information that you got from tagmate.io SaaS widget
